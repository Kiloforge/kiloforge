# Product Definition

## Project Name

kiloforge

## Description

A local development forge that runs the Cortex control plane, Command Deck, and Claude Code swarms directly on your machine. Transforms pure intent into meaningful action by coordinating AI agent swarms for parallel development at scale.

## Problem Statement

Coordinating multiple AI agents across multiple projects demands infrastructure that is observable, automated, and under your control. There's no easy way to run parallel Claude Code agents with quota tracking, merge coordination, and real-time observability without building custom infrastructure.

## Target Users

Kiloforgers — developers who use Claude Code agent swarms for parallel development and want full control over their local infrastructure.

## Key Goals

1. **Session management** — View logs, halt agents, and resume their Claude sessions interactively via the Command Deck
2. **Easy project onboarding** — Register projects and spawn agent swarms with minimal configuration
