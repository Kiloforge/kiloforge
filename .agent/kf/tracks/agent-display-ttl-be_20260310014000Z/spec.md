## Summary

Finished agents (completed, failed, stopped, force-killed) persist on the dashboard indefinitely, cluttering the active view and inflating status counts. Add a `finished_at` timestamp to the agent model, support filtering active vs. historical agents in the API, and add a dedicated history endpoint for the full agent list.

## Context

The dashboard currently shows ALL agents regardless of status. After extended use, the agent grid fills with completed/failed agents that obscure the currently running ones. The status bar histogram counts all agents, making it hard to see at a glance how many are actually active.

Users need:
1. **Dashboard view** — only active agents + recently finished (within ~30 minutes)
2. **History page** — all agents in a scrollable list, including long-finished ones
3. **Status bar** — only count active agents (running, waiting)

### FinishedAt population

In the spawner's `monitorAgent` goroutine (or wherever status transitions happen), set `FinishedAt` when status becomes terminal:

```go
func (s *Spawner) monitorAgent(agentID string, ...) {
    // ... wait for process exit ...
    now := time.Now()
    s.store.UpdateAgent(agentID, func(a *domain.AgentInfo) {
        a.Status = finalStatus
        a.FinishedAt = &now
        a.UpdatedAt = now
    })
}
```

### Active filter logic

```go
func (h *APIHandler) ListAgents(ctx context.Context, req gen.ListAgentsRequestObject) (...) {
    agents := h.agents.Agents()
    if req.Params.Active == nil || *req.Params.Active {
        cutoff := time.Now().Add(-30 * time.Minute)
        agents = filterActiveAgents(agents, cutoff)
    }
    // convert and return
}

func filterActiveAgents(agents []domain.AgentInfo, cutoff time.Time) []domain.AgentInfo {
    var result []domain.AgentInfo
    for _, a := range agents {
        if a.IsActive() {
            result = append(result, a)
        } else if a.FinishedAt != nil && a.FinishedAt.After(cutoff) {
            result = append(result, a)
        } else if a.FinishedAt == nil && a.UpdatedAt.After(cutoff) {
            // fallback for agents without FinishedAt (pre-migration)
            result = append(result, a)
        }
    }
    return result
}
```

### IsActive helper on domain

func (a AgentInfo) IsActive() bool {
