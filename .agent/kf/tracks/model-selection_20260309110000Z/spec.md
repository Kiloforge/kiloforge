# Specification: Configurable Model Selection with Opus Default

**Track ID:** model-selection_20260309110000Z
**Type:** Feature
**Created:** 2026-03-09T11:00:00Z
**Status:** Draft

## Summary

Add model selection configuration so agents can be spawned with a specific Claude model. Default to `opus` (latest Claude Opus). The model flows through config → spawner → `claude --model <model>` flag.

## Context

Currently the spawner invokes `claude -p <prompt> --session-id <id> --output-format stream-json` without specifying a model, which uses whatever Claude Code's own default is. Users should be able to configure which model their agents use — defaulting to `opus` for maximum capability. The Claude CLI supports `--model <alias>` where aliases like `opus`, `sonnet`, `haiku` resolve to the latest version of that model family.
