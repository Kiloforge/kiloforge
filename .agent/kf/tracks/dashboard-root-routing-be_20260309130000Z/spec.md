The dashboard should be the primary entry point. Gitea is a backing service that users rarely access directly — especially once reverse proxy auth is enabled (see `gitea-proxy-authn_20260309123000Z`).

Additionally, the default admin username is still "conductor" (pre-rebrand), the Gitea container is named "conductor-gitea", and the dashboard has no favicon.

### Gitea proxy with subpath stripping
```go
func NewGiteaProxy(targetURL, authUser string) http.Handler {
    target, _ := url.Parse(targetURL)
    proxy := httputil.NewSingleHostReverseProxy(target)
    return http.StripPrefix("/gitea", http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        r.Header.Set("X-WEBAUTH-USER", authUser)
        proxy.ServeHTTP(w, r)
    }))
}
```
