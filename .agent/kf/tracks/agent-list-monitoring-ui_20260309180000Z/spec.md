# Specification: Agent List with Role/Track Links and Monitoring View

**Track ID:** agent-list-monitoring-ui_20260309180000Z
**Type:** Feature
**Created:** 2026-03-09T18:00:00Z
**Status:** Draft

## Summary

Enhance the dashboard agent grid with clickable track/ref links, add a dedicated agent detail page at `/agents/:id` with embedded log viewer and terminal, and add role/status filter chips to the overview page.

## Context

The overview page already displays an agent grid via `AgentCard` components showing role, status, tokens, model, uptime, and a `ref` field. However, the `ref` is plain text (not linked), there's no dedicated agent detail page (logs and terminal are only accessible via modals), and there's no way to filter agents by role or status.

## Codebase Analysis

### Existing components

- **`frontend/src/pages/OverviewPage.tsx`** — Agent grid sorted by status, fetches from `GET /api/agents`
- **`frontend/src/components/AgentCard.tsx`** — Card with role badge, status, tokens, cost, ref (plain text), "View Log" and "Attach" buttons
- **`frontend/src/components/LogViewer.tsx`** — Log viewer with follow mode, fetches from `GET /api/agents/{id}/log`
- **`frontend/src/components/AgentTerminal.tsx`** — WebSocket terminal for interactive agents at `/ws/agent/{id}`
- **`frontend/src/App.tsx`** — Router with `/`, `/projects/:slug`, `/traces/:traceId` routes. No `/agents/:id` route.

### API endpoints available

- `GET /api/agents` — Returns all agents with role, ref, status, model, tokens, etc.
- `GET /api/agents/{id}` — Single agent detail
- `GET /api/agents/{id}/log?lines=200&follow=true` — Log with optional SSE streaming

### Agent data model

- `id`, `role` (developer/reviewer/interactive), `ref` (track ID or label like "track-gen"), `status` (11 states), `session_id`, `pid`, `worktree_dir`, `log_file`, `started_at`, `updated_at`, `model`

### Track-agent linking

- Board cards have `agent_id` and `track_id` fields — can cross-reference
- Agent `ref` field typically contains the track ID for developer agents
