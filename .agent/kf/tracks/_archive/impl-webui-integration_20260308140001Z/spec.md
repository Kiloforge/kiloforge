## Summary

Integrate the web dashboard into the kiloforge CLI lifecycle: add `dashboard_port` config, optionally start dashboard alongside relay in `kf up`, add standalone `kf dashboard` command, and wire Gitea links into the UI.

## Context

The dashboard server (track `impl-webui-server_20260308140000Z`) exists as a standalone package but isn't wired into the CLI or startup lifecycle. This track makes it accessible to users through the existing CLI commands and configuration system.

## Codebase Analysis

- **`kf up`**: `internal/cli/up.go` — starts Gitea via compose, then runs relay server in foreground
- **Config**: `internal/config/` — port/adapter resolution with defaults, JSON, env, flags
- **Relay server**: runs on `RelayPort` (3001), blocks on `ListenAndServe()`
- **Dashboard server**: `internal/dashboard/` (from track 1) — runs on its own port, needs store + tracker

### Integration points

1. **Config**: add `DashboardPort` field (default: 3002), `DashboardEnabled` bool (default: true)
2. **`kf up`**: start dashboard server in a goroutine alongside relay server
3. **`kf dashboard`**: standalone command to start only the dashboard (for when relay is already running)
4. **`kf status`**: print dashboard URL when running

## Technical Notes

### `kf up` integration

```go
// In up.go Run function:

// Start dashboard if enabled
if cfg.DashboardEnabled {
    dashSrv := dashboard.New(cfg.DashboardPort, store, tracker, cfg.GiteaURL())
    go func() {
        if err := dashSrv.Run(ctx); err != nil && err != http.ErrServerClosed {
            fmt.Fprintf(os.Stderr, "dashboard error: %v\n", err)
        }
    }()
    fmt.Printf("Dashboard:   http://localhost:%d\n", cfg.DashboardPort)
}

// Start relay server (blocking)
relaySrv := relay.NewServer(cfg, registry, cfg.RelayPort)
return relaySrv.Run(ctx)
```

### `kf dashboard` command

```go
var dashboardCmd = &cobra.Command{
    Use:   "dashboard",
    Short: "Start the web dashboard (standalone)",
    RunE: func(cmd *cobra.Command, args []string) error {
        cfg, _ := config.Resolve()
        store, _ := state.Load(cfg.DataDir)
        tracker := agent.NewQuotaTracker(cfg.DataDir)
        tracker.Load()
        srv := dashboard.New(cfg.DashboardPort, store, tracker, cfg.GiteaURL())
        fmt.Printf("Dashboard running at http://localhost:%d\n", cfg.DashboardPort)
        return srv.Run(ctx)
    },
}
```
