## Summary

Adding or removing a project via the dashboard doesn't update the project list in real time. The backend correctly emits `project_update` and `project_removed` SSE events, and the `useProjects` hook already has cache-invalidation handlers, but `App.tsx` never registers them with the SSE subscription.

## Context

The SSE subscription in `App.tsx` only registers handlers for `agent_update`, `agent_removed`, `quota_update`, `track_update`, and `track_removed`. The `project_update` and `project_removed` events are dropped because no handler is registered. The `useProjects` hook already exports `handleProjectUpdate` and `handleProjectRemoved` callbacks that correctly update the TanStack Query cache — they just need to be wired in.
