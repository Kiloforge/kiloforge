## Technical Notes

### Button Visibility Logic
```
Running/Waiting → Show: Stop
Stopped/Completed/Failed → Show: Resume (interactive only), Delete
```

### Mutation Hooks
Create a `useAgentActions` hook (or inline mutations) using TanStack Query `useMutation`:
- `stopAgent(id)` → `POST /api/agents/{id}/stop`
- `resumeAgent(id)` → `POST /api/agents/{id}/resume` → returns agent with ws_url
- `deleteAgent(id)` → `DELETE /api/agents/{id}`

On success, invalidate `queryKeys.agents()` and `queryKeys.agent(id)`. SSE will also push updates so this is belt-and-suspenders.

### Confirmation Dialog
Use a simple confirm dialog for Delete (browser `window.confirm` or a styled modal). Stop doesn't need confirmation — it's easily reversible via Resume.
