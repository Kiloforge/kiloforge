## Summary

Investigate replacing the raw `docker run` approach in `internal/gitea/manager.go` with a docker-compose based lifecycle, ensuring compatibility with both `docker compose` (v2 plugin) and `docker-compose` (v1 standalone, needed for Colima).

## Context

The current `kf init` command starts Gitea via a hardcoded `docker run` command with ~10 flags (ports, volumes, env vars). This is fragile, hard to extend (e.g., adding services like a reverse proxy), and doesn't leverage docker-compose's declarative model. Colima users on macOS often have only the v1 `docker-compose` binary, so both CLI variants must be supported.

## Codebase Analysis

- **`internal/gitea/manager.go:Start()`** — Currently builds a `docker run` command with flags for image, ports, volumes, env vars, container name. Checks existing container state before creating.
- **`internal/gitea/manager.go:WaitReady()`** — Polls Gitea API after container start. Independent of how container is started.
- **`internal/gitea/manager.go:Configure()`** — Creates admin user, token, repo via REST API. Independent of container lifecycle.
- **`internal/cli/init.go`** — Orchestrates the full init flow: Start -> WaitReady -> Configure -> SetupGitRemote -> start relay.
- **`internal/cli/destroy.go`** — Tears down via `docker rm -f`. Would need updating.
- **`internal/cli/status.go`** — Checks container via `docker inspect`. Would need updating.
- **No docker-compose.yml exists** in the project.

## Technical Notes

Key areas to investigate:
1. **CLI detection** — `docker compose version` vs `docker-compose version`, fallback order
2. **Compose file generation** — Should kiloforge generate a `docker-compose.yml` at init time or ship a template?
3. **Volume and network management** — Named volumes vs bind mounts, custom networks
4. **Compose file location** — In project dir, in `~/.kiloforge/`, or in data dir?
5. **`docker compose up -d` vs `docker compose start`** — Lifecycle semantics
6. **Colima quirks** — `host.docker.internal` support, volume mount performance, known issues
