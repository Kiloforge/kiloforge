### Why Apache 2.0

- Permissive: allows commercial use, modification, distribution
- **Requires attribution**: derivative works must retain copyright notices, state changes made, and include the license
- Patent grant: includes explicit patent license (MIT does not)
- NOTICE file convention: standard place to list upstream attributions
- Compatible with MIT: MIT-licensed code can be included in Apache 2.0 projects
