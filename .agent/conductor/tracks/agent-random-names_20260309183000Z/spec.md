## Summary

Generate a random, memorable, slightly silly human-friendly name for each agent on spawn. Format: `"{adverb} {adjective} {name}"` — e.g., "curiously dangerous dave", "remarkably cheerful grace", "suspiciously quiet oscar". The ID remains the system identifier for uniqueness; the name is for human readability in the UI and logs.

## Context

Agents currently display as short UUID hashes (e.g., `dev-abc123f`) in the dashboard. When multiple agents are running, it's hard to distinguish them at a glance. A memorable random name makes it easy to say "check on 'suspiciously quiet oscar'" instead of "check on dev-abc123f".
