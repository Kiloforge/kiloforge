## Technical Notes

### Auto-sync on empty board fetch

Add to `GetBoard` handler — if board has zero cards, trigger a sync before returning:

```go
func (h *APIHandler) GetBoard(ctx context.Context, req gen.GetBoardRequestObject) (...) {
    board, err := h.boardSvc.GetBoard(req.Project)
    if err != nil { return error response }

    // Auto-sync if board is empty (first load or after reset)
    if len(board.Cards) == 0 {
        projectDir := h.resolveProjectDir(req.Project)
        if projectDir != "" {
            tracks, _ := service.DiscoverTracks(projectDir)
            if len(tracks) > 0 {
                result, err := h.boardSvc.SyncFromTracks(req.Project, tracks, nil)
                if err == nil {
                    board, _ = h.boardSvc.GetBoard(req.Project) // re-read after sync
                    // publish SSE event
                }
            }
        }
    }
    return board
}
```

This is a lightweight check — only syncs when board is empty, not on every fetch.
