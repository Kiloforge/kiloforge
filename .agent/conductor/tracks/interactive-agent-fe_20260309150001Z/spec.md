### Markdown rendering
Agent output often includes code blocks, lists, and formatting. Use a lightweight markdown renderer (e.g., `react-markdown` or similar) for agent messages. User messages render as plain text.

### WebSocket message handling
```typescript
ws.onmessage = (event) => {
  const msg = JSON.parse(event.data);
  switch (msg.type) {
    case "output": // agent text → append to messages
    case "status": // agent lifecycle → update status indicator
    case "error":  // error → show error banner
  }
};

// Send user input
ws.send(JSON.stringify({ type: "input", text: userText }));
```
